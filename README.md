![Image Of Dr. Mario](https://upload.wikimedia.org/wikipedia/en/f/f8/Dr._Mario_box_art.jpg)

## User Guide

### Run all projects

`./mvnw clean package`

### Run single project

`./mvnw -pl <project-name> clean package`

### Make release

`./script/make-release.sh <new-version> <next-snapshot-version> <new-branch-name>`
